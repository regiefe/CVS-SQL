# csv_sql
A partir do csv criada as respectivas tabelas e as relaciona no arquivo .db 

![csv to sql](img/demo.png "demo tabela criada e relacionada")

## Instalação

----

```
git clone git@github.com:regiefe/db-sql.git

```
## Mode de usar

```
cd csv_sql/App
./parser_cvs [ caminho do .csv  ]

```
