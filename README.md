# csv_sql
gera arquivo  sql a partir de um arquivo cvs

![csv to sql](img/parser_cargo.png "Lê csv e gera tabela sql")

## Instalação

----

```
git clone git@github.com:regiefe/db-sql.git

```
## Mode de usar

```
cd csv_sql/App
./parser_cvs [ caminho do .csv  ]

```
